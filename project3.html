Project 3: ROM-Based Phone Sorter

This tool sorts a dataset of mobile phones based on ROM size using the QuickSort algorithm. ROM sizes are converted into megabytes to normalize various units like GB, MB, KB, and non-numeric values like "UpTo" or "Expandable." Secondary sorting by attributes such as rating, price, or RAM is also supported. Sorting is interactive and outputs a CSV.
