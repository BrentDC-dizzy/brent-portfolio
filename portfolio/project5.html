Project 5: Grade Analyzer with Sorting and Chart

This Python application allows users to load a CSV of student grades and perform sorting operations using Merge Sort. It features a GUI built with Tkinter, interactive buttons for sorting scores in ascending or descending order, and a bar chart visualizer to display grade distribution. Sorted results are exportable to CSV, and all visual outputs are customizable and savable.
