Project 6: Quicksort-Only LLM (Locally Hosted LLaMA 3.3 70B)

This project features a locally hosted instance of LLaMA 3.3 70B Instruct Turbo, fine-tuned to answer only questions related to the Quick Sort algorithm. Any off-topic questions are gracefully rejected with a custom refusal response.

Built using Flask for the backend and a responsive frontend using HTML, CSS, and JavaScript, the app includes a dynamic sidebar for managing chat history. Users can create new chats, delete old ones, and switch between light and dark themes seamlessly.

Chat sessions are saved locally using localStorage, ensuring conversations persist across sessions. This project demonstrates prompt engineering, model safety enforcement, and an intuitive UI tailored for algorithm education.
